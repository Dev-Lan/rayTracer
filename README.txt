I believe I have completed all parts to this assignment, excluding the extra credit.

I compiled on my mac machine using the provided makefile. I will test on lab machines
shortly.

To COMPILE I simply run "make" in the same directory as this README.txt

To RUN I call "./run <inputfile>"" for example:
	
	./run testInput.txt

The output will be put in the same directory as this README.txt file
and will be named the same as the input file with a .ppm extension.

The resulting file will be placed in this directory even if the input file
is located elsewhere.
